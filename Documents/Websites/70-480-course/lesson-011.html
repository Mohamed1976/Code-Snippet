In 2007, Jeff Atwood made the quote that was popularly referred to as Atwood's Law: “Any application that can be written in JavaScript, will eventually be written in JavaScript.”

Node.js runtime is an alternative for server side code such as c#, it allows you to interact with database and files. Node.js can be used for example to build webservices. DOM (Document Object Model) is the functions and objects that JavaScript has access to in the webbrowser.

In JavaScript there are three ways to declare variables:
  • let Use it in loops
  • var Available in the entire scope, could lead to mistakes being made.
  • const Use it by default, clearly says this variable does not get reassigned
Note: const and let can cover almost any usercase.
