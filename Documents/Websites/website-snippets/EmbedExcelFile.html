How to Embed an Excel File on a Web Page using plain HTML.

Introduction

Using the Microsoft API allows the embedding of Excel files very easily online and gives many features. It is not recommended to do it any other way. And the code is extremely simple. It is shown below.

<iframe src='https://view.officeapps.live.com/op/embed.aspx?src=specify path here' width='100%' height='565px' frameborder='0'> </iframe>

Microsoft Office Viewer

Only files under 10 MB can be previewed with the Microsoft Office Viewer.

  • Office Viewer Link: https://view.officeapps.live.com/op/view.aspx?src=[OFFICE_FILE_URL]
  • Office Viewer Embedded Link: https://view.officeapps.live.com/op/embed.aspx?src=[OFFICE_FILE_URL]

Microsoft Office Viewer only previews office document types, listed below:

  1. Microsoft Word (.DOCX, .DOCM, .DOTM, .DOTX, .DOC)
  2. Microsoft Excel (.XLSX, .XLSB, .XLS, .XLSM)
  3. Microsoft PowerPoint (.PPTX, .PPSX, .PPT, .PPS, .PPTM, .POTM, .PPAM, .POTX, .PPSM)
